# ExportHub 3D — Global Trade, Reimagined

ExportHub is a premium, futuristic 3D website built for an import-export business connecting manufacturers in India with global buyers across the world.

## Live Demo
Once GitHub Pages is enabled, you can view the live site here:
👉 **[https://shrishkumar111222.github.io/exporthub/](https://shrishkumar111222.github.io/exporthub/)**

---

## 🚀 Key Visual & Interactive Features

1. **Hero Section**: 3D realistic rotating Earth globe, stars particle background, day/night lighting, and flowing trade lines extending from India.
2. **Trade Network**: Focuses on major trade corridors (Russia, UAE, Germany, USA, Brazil, Japan). Interactive pins display customs documentation, shipping times, and certifications on click.
3. **Container Port**: Scroll-driven 3D port scene. Moving containers transfer from warehouse shelves onto the ocean vessel as you scroll.
4. **Futuristic Warehouse**: Interactive shelves categorizing products into Medical, Food, Agriculture, Industrial, and Electronics.
5. **360° Product Viewer**: Allows interactive 3D orbit rotation and inspection of procedural models (Syringes, Nitrile Gloves, Masks, Basmati Rice, etc.).
6. **Logistics Spline Journey**: Synced scroll animation following a shipment's exact tracking path.
7. **AI Trade Assistant**: Holographic interactive chatbot with automated compliance recommendations, HS code lookups, and shipping cost quotes.
8. **Company Stats & Testimonials**: Dynamic counting animations and a glassmorphic testimonials slider.

## 🛠️ Built With

- **Three.js** (WebGL 3D rendering)
- **GSAP & ScrollTrigger** (Custom timeline and scroll animations)
- **HTML5 / CSS3** (Futuristic glassmorphic design and responsive layout)

## 📦 How to Run Locally

1. Clone this repository:
   ```bash
   git clone https://github.com/shrishkumar111222/exporthub.git
   ```
2. Navigate to the project directory and start a local HTTP server:
   ```bash
   # Using Python
   python3 -m http.server 8080
   
   # Or using Node/NPM
   npx live-server
   ```
3. Open `http://localhost:8080` in your web browser.

## 🌐 Deploy to GitHub Pages (Free Hosting)

To make the website publicly viewable:
1. Go to your repository page on GitHub: `https://github.com/shrishkumar111222/exporthub`.
2. Click **Settings** (gear icon) on the top menu bar.
3. Under the **Code and automation** section on the left sidebar, click **Pages**.
4. Under **Build and deployment**:
   - **Source**: Select `Deploy from a branch`.
   - **Branch**: Select `main` and the folder `/ (root)`.
5. Click **Save**.
6. Within a minute, GitHub will give you the live link (e.g. `https://shrishkumar111222.github.io/exporthub/`) to share with anyone!
